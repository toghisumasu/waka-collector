## 現在の実装状態

### ShikimokuChecker#next_constraints（新規追加）

```ruby
checker.next_constraints(history, bui_dict: nil)
# => {
#      verse_type:    :chouku | :tanku,
#      forbidden_bui: ["降物", "光物", ...],
#      season_hint:   { current: "秋", count: 2,
#                       must_continue: true, must_switch: false }
#    }
```

- `verse_type`: 直前句の長短を反転
- `forbidden_bui`: 句去ルールから次句で違反になる部立を逆算
- `season_hint`: kukazo_rules の min/max から継続・転季の義務を判定

### RengaGenerator の変更点

- `initialize` に `constraints:` キーワード引数を追加
- `filter_pool` に `must_continue` / `must_switch` による季節絞り込みを追加
- 従来の前句季語からの推定はフォールバックとして残存

### RengasController の変更点

- `build_verse_history` private メソッドを追加（直近9句を遡って verse Hash 化）
- `season_from_text` private メソッドを追加（句テキストから季節を推定）
- `next_constraints` → `RengaGenerator` への配管を完成
