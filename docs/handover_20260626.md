### 主要メソッド早見表

```ruby
checker = ShikimokuChecker.new

# 単体チェック
checker.kuzari_ok?(history_bui, candidate_bui)
checker.kukazo_ok?(history, candidate)
checker.chotan_ok?(history, candidate)

# 統合チェック（推奨）
checker.all_violations(history, candidate, bui_dict: bui_dict)
checker.scan_chain_with_chotan(chain, bui_dict: bui_dict)

# 定座チェック
checker.teiza_tsuki_violations(chain, faces)
checker.teiza_hana_violations(chain, folds)

# ユーティリティ
ShikimokuChecker.describe(violation)  # 人間可読テキスト
```

### verse Hash のフォーマット

```ruby
{
  word:       "雲にけふ",        # 句の冒頭（bui_dict キーと一致させると taiyo 参照可能）
  bui:        ["聳物","山類","植物"],
  season:     "春",              # "春"/"夏"/"秋"/"冬"/"恋"/"雑"/nil
  verse_type: :chouku,           # :chouku（5-7-5）/ :tanku（7-7）
  tsuki:      true,              # 月の定座用（省略可）
  hana:       true,              # 花の定座用（省略可）
  plant_type: :flower,           # :flower/:grass/:tree（植物句のみ）
}
```


**追加候補（連歌頻出・誤分割リスクあり語）:**

| 語 | 読み | 追加理由 |
|------|------|---------|
| 蓬生 | よもぎふ | 複合語、単純分割される恐れ |
| 薄花 | うすはな | 「うす」+「花」に分割される |
| 小萩 | こはぎ | 「小」+「萩」に分割される |
| 荒田 | あらた | 「荒」+「田」に分割される |
| 上風 | うわかぜ | 「上」+「風」に誤読の恐れ |
| 霞立つ | かすみたつ | 動詞複合で聳物を正確に認識するため |

ユーザー辞書再ビルド手順:
```bash
/usr/lib/mecab/mecab-dict-index \
  -d /var/lib/mecab/dic/ipadic \
  -u dict/user.dic \
  -f utf-8 -t utf-8 dict/user_entries.csv
```

#### C. DB 連携インターフェース設計

`ShikimokuChecker` はすでに Rails/MeCab 非依存の純 Ruby。

```ruby
history   = Renga.chain.map { |r| { bui: r.bui_list, season: r.season, plant_type: r.plant_type } }
candidate = { bui: nlp_result.bui_list, season: nlp_result.season, plant_type: nlp_result.plant_type }
violations = checker.all_violations(history, candidate, bui_dict: BuiDictionary.new)
```
